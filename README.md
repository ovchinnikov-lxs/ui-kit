## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint/Fix with ESLint and StyleLint

```sh
npm run lint:check
npm run lint:fix
```
